# Reference Materials

This folder keeps the source idea artifacts that guided the prototype:

- `US_SCAA_main.tex`: LaTeX draft for the SCAA paper idea.
- `US_SCAA_paper.pdf`: compiled paper draft.
- `advisor_note.jpg`: advisor message describing the intended direction.
- `final_note.txt`: short note from the idea folder.

The implementation in this repository follows the practical V1 version of that
idea: reproduce the yield model, detect detrended low-yield anomalies, attribute
each anomaly to sparse feasible weather changes, and add a baseline for early or
mid-season anomaly warning.
